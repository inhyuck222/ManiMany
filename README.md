# 마니 매니


도서관을 더 자주 이용하는 학생들을 위해 만들어진 도서관 이용 웹 서비스는 ‘마니 매니 라이브러리’ 이다.

이 프로그램은 도서관을 자주 이용하는 학생들에게 언제 어디서든 도서관을 이용하도록 지원한다. 관리자들은 회원관리와 홍보 업무를 통해 도서관 이용률을 증가 시킨다.

이 프로그램은 다른 도서관 이용 서비스와 달리 알림 서비스를 제공한다.
